# Knowledge Base Search

Search organization documents and CRM data for answers.

## Tools

Uses MCP tool from `silocrm` server:
- `silocrm.knowledge.search` - Vector search across documents and CRM

## Content Sources

### Documents (User-uploaded)
- SOPs and playbooks
- Product information
- Training materials
- Process documentation

### CRM Data (Auto-indexed)
- Contact notes
- Call transcripts
- Lead descriptions

## Usage

When users ask about:
- "How do we handle..." - Search knowledge base
- "What's our process for..." - Search documents
- "Do we have documentation on..." - Search knowledge base

## Response Format

1. Search knowledge base first
2. Cite document sources
3. Synthesize answer from results
4. Offer to search more if needed
