# Skills

## Semantic Search

Use `semantic_search` for natural language queries about CRM data:
- "Find leads who mentioned budget concerns"
- "Show contacts interested in enterprise plans"
- "Any leads worried about installation time"

### Best Practices
1. Use descriptive, natural language queries
2. Specify types filter if you only need leads, contacts, or notes
3. Results include similarity scores - higher is better match

### Formatting Rules
- Text formatting: Use plain text only, NO markdown
- Present results clearly with name, type, and relevance
