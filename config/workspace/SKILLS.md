# Skills

## CRM Tools via MCP

All CRM tools are available through the **silocrm** MCP server. Call `tools/list` to discover available tools with their descriptions and schemas.

### Usage Guidelines

1. **Read operations** - Use freely to answer user questions about leads, contacts, pipeline, KPIs, etc.

2. **Write operations** - Always confirm with user before:
   - Creating reminders/tasks
   - Booking appointments
   - Creating workflows (saved as INACTIVE)
   - Creating sequences

3. **Charts & Visualizations** - Generate charts when presenting analytics or reports.

4. **Date ranges** - Use YYYY-MM-DD format for date parameters.
